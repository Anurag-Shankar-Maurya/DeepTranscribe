How to Get a Deepgram API Key

  1. Go to the Deepgram Console and create an account
  2. After signing in, click on "API Keys" in the sidebar
  3. Click "Create API Key" and give it a name (e.g., "DeepTranscribe")
  4. Copy the API key that's generated
  5. Add the key to your environment variables as DEEPGRAM_API_KEY

Free Tier Limitations
