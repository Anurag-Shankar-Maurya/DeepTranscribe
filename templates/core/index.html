Developer Friendly API

Need to integrate transcription into your own app? DeepTranscribe provides secure API endpoints with token authentication.

GET /api/transcripts/ POST /api/chat/ 
import requests

token = "your_api_token"
files = {'audio': open('meeting.mp3', 'rb')}

response = requests.post(
    "https://deeptranscribe.com/api/upload/", 
    headers={"Authorization": f"Token {token}"},
    files=files
)

print(response.json()['transcript_id'])
